# AisToXmlRouteConvertor

## Overview
AisToXmlRouteConvertor converts AIS (Automatic Identification System) route data into XML route files compatible with navigation or routing systems.

## Prerequisites
- Access to AIS route data files (CSV, JSON, or supported formats)
- A machine with the application installed

## Step-by-Step Instructions
### Import AIS Data
1. Open the application.
2. Use the `Import` function to select your AIS data file.
3. Verify the data preview and mapping.
4. Click `Convert` to produce the XML route file.
5. **Expected Result**: An XML file is created and saved to the selected output folder.

## Tips and Best Practices
- Validate AIS data for completeness before converting.
- Keep backups of original files.
