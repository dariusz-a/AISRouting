# Target Users

## Overview
This section describes the intended users of AisToXmlRouteConvertor.

## Prerequisites
- Basic familiarity with AIS data formats (CSV/JSON)
- Access granted by QA leads (internal tool)

## Step-by-Step Instructions
### Using the Tool as a QA Engineer
1. Obtain AIS data sample files from the test dataset.
2. Launch the tool and select the sample file for conversion.
3. Verify output XML against expected route templates.

## Tips and Best Practices
- Keep the tool's codebase simple to ease maintenance and debugging.
- Use version-controlled test datasets for repeatable results.
