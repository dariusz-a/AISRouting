# MyVessels User Manual Q&A Session

## Q1: Primary user types and outcomes
**Question:** Who are the primary user types (roles) of MyVessels (e.g., fleet manager, vessel operator, port authority, dispatcher, analyst), and what are the top 1–2 outcomes each role needs to achieve in the system?

**Answer:** fleet manager

## Q2: Fleet Manager key outcomes
**Question:** For the Fleet Manager role, what are the top 1–2 outcomes they must achieve in MyVessels? For example, do they primarily need real-time fleet position monitoring, automated alerts for deviations (geofences/ETA delays), voyage oversight and status, or compliance/reporting visibility?

**Answer:** they primarily need real-time fleet position monitoring
